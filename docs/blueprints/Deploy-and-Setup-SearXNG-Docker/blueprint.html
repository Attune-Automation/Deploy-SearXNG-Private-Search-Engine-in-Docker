Description

This is an Attune Project to deploy and startup the SearXNG Private Search Engine in Docker. SearXNG is a free internet metasearch engine which aggregates results from more than 70 search services.

Step 1: Clone the SearXNG Docker Code

Clone the SearXNG Docker code from GitHub. 


# parse Attune Parameters into script varaibles. Not required however, this 
# can assist with debugging and maintaining the scripts.
baseDirectory={searxngSearchEngineBaseDirectory}

# If it doesn't exist create the base directory and change to that directory.
mkdir -p "$baseDirectory" && cd "$_"

# Clone the SearXNG Docker code from GitHub.
git clone https://github.com/searxng/searxng-docker.git

# Log the files that were deployed. Not required however, capturing logs is 
# good practice.
ls -ltrh searxng-docker/

Step 2: Generate Secret Key and Update settings.xml

Clone the SearXNG Docker code from GitHub. 


baseDirectory={searxngSearchEngineBaseDirectory}

cd $baseDirectory/searxng-docker/

# Update the settings file with a generated secret key.
sed -i "s|ultrasecretkey|$(openssl rand -hex 32)|g" searxng/settings.yml

# log that the secret key was updated.
less searxng/settings.yml | grep secret_key
